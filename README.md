# Golang Todo CLI

一個簡單的 Golang 指令列待辦清單工具，支援新增、查看、完成、刪除任務，並將資料儲存在 `todos.json` 檔案中，讓你每次開啟程式都能延續上次進度。

---

## 功能特色

- 查看所有待辦事項
- 新增任務
- 標記任務為完成
- 刪除任務
- 自動儲存至 `todos.json`

---

## 使用方式

### 安裝並執行

```bash
git clone https://github.com/samuel361592/golang-todo-cli.git
cd golang-todo-cli
go run main.go
```
Go 會自動讀取或建立 todos.json 檔案來儲存資料
